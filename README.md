# investicia
Описание
Игра представляет собой симулятор инвестиционных решений, предназначенный для тех, кто хочет научиться основам финансового планирования и повысить свою финансовую грамотность. Игроки смогут попробовать себя в роли инвестора, управляя виртуальным капиталом и принимая инвестиционные решения в условиях постоянно меняющегося рынка.

Цель игры:
Игрок должен сформировать успешный инвестиционный портфель, учитывая риски и возможную прибыль. Чем больше капитала удастся заработать игроку за установленное игровое время, тем выше его итоговая позиция в рейтинге участников.

Ключевые особенности игры:
Разнообразие активов: акции компаний разных секторов экономики, государственные облигации, сырьевые товары и криптовалюты.
Реалистичные рыночные условия: игроки сталкиваются с ситуациями, близкими к реальной экономике, включая кризисы, экономические спады и периоды роста.
Регулирование риска: игрок может управлять уровнем риска своего портфеля, выбирая подходящие активы и стратегию диверсификации.
Обучение финансовой грамотности: игрокам предоставляются базовые знания по финансовым инструментам, экономическому анализу и управлению капиталом.
Конкуренция игроков: каждый участник играет одновременно с другими пользователями, создавая динамичную среду конкуренции.
Как играть?
Пользователь выбирает начальные инвестиции, распределяет средства среди выбранных активов и следит за изменениями рыночной ситуации. Каждый раунд длится определенное количество игровых месяцев, по истечении которых подводятся итоги раунда и определяется победитель.
